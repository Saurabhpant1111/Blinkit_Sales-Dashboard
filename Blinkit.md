# Blinkit Sales Analysis Dashboard

## Project Overview

This project is an interactive Power BI dashboard built to analyze Blinkit sales performance, customer behavior, product performance, and outlet insights.

The dashboard helps in tracking key business metrics and identifying trends through interactive visualizations.

## Tools Used

* SQL
* Power BI
* DAX
* Power Query

## Key KPIs

* Total Sales
* Total Orders
* Average Rating
* Total Customers

## Dashboard Features

* Sales by Item Type Analysis
* Outlet Performance Analysis
* Sales by City
* Monthly Sales Trend
* Top Customers Analysis
* Interactive Filters and Slicers

## SQL Analysis Performed

* Top 5 Customers by Sales
* Sales by Item Type
* Outlet Performance Analysis
* City-wise Sales Analysis
* Monthly Sales Trend Analysis
* Rating-based Performance Analysis
